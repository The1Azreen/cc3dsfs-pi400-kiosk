# cc3dsfs-Pi400-Kiosk Setup

This repository automates the installation and kiosk configuration of [cc3dsfs](https://github.com/Lorenzooone/cc3dsfs) on a **Raspberry Pi 400**.  
It compiles cc3dsfs, configures the Pi for fullscreen kiosk mode, and ensures the program auto-restarts if it ever crashes.
